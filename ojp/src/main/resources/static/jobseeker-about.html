About JobHuntPro

JobHuntPro is a modern, user-friendly job portal developed to bridge the gap between job seekers and employers. It is designed to replicate real-world platforms like Naukri, LinkedIn Jobs, and Indeed with added simplicity and focused features.

It includes features like:

  • Role-based authentication for Admin, Job Seekers, and Recruiters
  • Advanced job filtering (by location, salary, skills, experience)
  • Resume upload, profile management, job posting, and application tracking
  • Admin dashboard to monitor platform activity and user analytics

This project is built using:

  • Backend: Java, Spring Boot, REST API, MySQL
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
